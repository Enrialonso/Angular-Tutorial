Directiva - NgClass

Agrega y elimina atributos CSS en un elemento HTML. segun la logica de programacion que tengamos del lado interno de nuestro componente podemos hacer que un elemento html cambie su CSS para ilustrale al usuario de alguna manera el dinamismo de la pagina. A continuacion mostramos como es la sintaxis de esta directiva

{{ textoDirectivaNgstyle }}

NgStyle - tipo Texto

Pasa el mouse sobre el elemento:
Elemento HTML
Activamos esta directiva con la ayuda de los event Binding en el mouse para cambiar el estilo de CSS aplicado al elemento, cuando el mouse esta sobre el elemento asignamos "font-style: italic" a la clase y observamos como el texto cambia su formato, cuando quitamos el mouse del elemento vuelve a la normalidad.
