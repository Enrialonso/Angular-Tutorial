Variables locales - NgFor

Esta directiva tiene distintas variables locales que son vastante utiles para poder facilitar el manejo y construccion de la estructura que queremos realizar con la directiva. son variables que funcionan de forma local dentro de bucle y las podemos utilizar para no depender de declaraciones externas del control del bucle.

  • Index: Number guarda el numero actual del ciclo que se esta ejecutando.
  • Frist: Boolean es verdadero si es el primer ciclo del bucle.
  • Last: Boolean es verdadero si es el ultimo ciclo del bucle.
  • Even: Booleanes verdadero si numero de ciclo es un numero par.
  • Odd: Boolean es verdadero si numero de ciclo es un numero impar.

Normalmente esta directiva es muy util para poder estructurar tablas, paneles de informacion o snips de html con informacion variada en formato de objeto. Para los ejemplos posteriores vamos a utilizar un objeto con la siguiente estructura: 

 Objeto: "Tienda"
                Tienda = [ { Producto(Libro), Precio: 200, } ]
