Property Binding

Property Binding

Property Binding es el segundo tipo de data binding empleados por Angular para enlazar valores de la fuente de datos a la vista. En este caso, se trata de un enlace que relaciona un atributo con una expresión, con la siguiente sintaxis:


{{ textoProperty }}
