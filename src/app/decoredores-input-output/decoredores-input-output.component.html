Decoradores Input/Output

Estos decoradores tiene la funcion de pasar datos entre componentes en ambos sentidos segun como se plantee la logica de los componentes que usan estos decoradores.

Se declaran mediante el empleo del decorador @input/Output. En los ejemplos sigientes pasaremos datos entre componentes y los llamaremos para la ocacion componentes Padre e Hijo.
