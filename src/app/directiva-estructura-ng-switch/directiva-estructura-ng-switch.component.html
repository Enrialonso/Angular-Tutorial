Directiva - NgSwitch

Como en mucho lenguajes de programacion para evitar el anidamiento infinito de sentencias "IF" existe la sentencia NgSwitch, la cual tiene como entrada una variable la cual sera evalueda reiteradamente segun los casos necesarios para la sentencia. dependiendo de la evaluacion renderizaremos una parte del HTML u otra, en caso de que no coincida con ninguna condicion tenemos una sentencia Default la cual renderiza el HTML cuando no hay ninguna coincidancia.

En esta sintaxis, se crea dentro del elemento html una variable local con let que recorrerá el array definido por of y proveniente del componente. Como buena práctica, el nombre de la variable local será singular y el del array plural.

{{ textoObjetoNgSwitch0 }}
{{ textoObjetoNgSwitch1 }}
{{ textoObjetoNgSwitch2 }}
{{ textoObjetoNgSwitch3 }}
{{ textoObjetoNgSwitch4 }}
