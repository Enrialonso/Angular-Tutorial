Event Binding - Concepto

Event Binding

Los enlaces event binding son enlaces de un solo sentido, one way, pero en este caso desde la vista a la fuente de datos, ya que como su nombre indica, los desencadena un evento en el cliente web.


Lista de Event Binding
  • focus: cuando un elemento recive el enfoque del mouse o cursor.
  • blur: cuando un elemento pierde el enfoque del mouse o cursor.
  • submit: cuando es presionado el boton submit de un form.
  • scroll: cuando se hace desplazamiento sobre elementos html con scroll control.
  • cut: cuando realizamos el comando cortar control + x sobre un elemento html.
  • copy: cuando realizamos el comando copiar control + c sobre un elemento html.
  • paste: cuando realizamos el comando pegar control + v sobre un elemento html.
  • keydown: cuando presionamos cualquier tecla del teclado.
  • keypress: cuando presionamos cualquier tecla del teclado exceptuando shift control alt
  • keyup: cuando se levanta una tecla presionada del teclado o se deja de pulsar la misma.
  • mouseenter: cuando el mouse entra en el area propia de un elemento de html.
  • mousedown: cuando el puntero del mouse esta sobre el elemento html y se realiza un click el evento dispara cuando el click es presionado.
  • mouseup: cuando el puntero del mouse esta sobre el elemento html y se realiza un click el evento dispara cuando el click es levantado o se deja de presionar.
  • click: cuando hacemos click sobre algun elemento html.
  • dbclick: cuando hacemos boble click sobre un elemento html.
  • drag: cuando desplazamos un elemento html.
  • dragover: cuando dejamos de desplazar un elemento html.
  • drop: cuando soltamos un elemento html que estavamos desplazando.
